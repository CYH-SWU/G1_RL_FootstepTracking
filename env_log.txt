说明价值网络没学到东西，策略更新将缺乏方向性。)

approx_kl	初期波动，后期稳定 < 0.02
(表示每次更新策略的变化幅度。
必须将其控制在目标 KL（通常 0.01~0.03）以下。
如果突然飙升到 0.1，说明更新过于激进，训练即将崩溃，应立即停止。)
合理范围通常在 0.005 到 0.02 之间，一个比较理想的区间是 0.008 到 0.013

def get_dampratio(joint_name: str) -> float:
    """根据关节名称返回推荐的 dampratio 值"""
    if "hip" in joint_name.lower():
        return 0.9
    elif "knee" in joint_name.lower():
        return 1.0
    elif "ankle" in joint_name.lower():
        return 1.2
    elif "waist" in joint_name.lower():
        return 0.9
    else:
        return 1.0
